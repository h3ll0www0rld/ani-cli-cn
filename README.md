# 安装
本项目使用`mpv`作为播放器，请确保`mpv`在你的环境变量中(在终端输入`mpv --version`有输出)
## Windows
```shell
scoop install mpv
```
## Ubuntu
```shell
sudo apt install mpv
```

# 开发&构建
## 开发
```shell
pip install -r requirements.txt
```
## 构建
### Windows
```powershell
./scripts/build.ps1
```
### Linux & Mac
```shell
./scripts/build.sh
```
